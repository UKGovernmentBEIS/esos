If you do not have an activity code for the undertaking, you are advised to select a standard industrial classification (SIC) code using the Office for National Statistic's ‘Summary of Structure worksheet’ in the following link: UK SIC 2007 - Office for National Statistics.

SIC codes are 5-digit codes used to classify economic activity, which are structured according to the formula: Division-Group-Class-(Sub-class). The digits for the economic activity which relate to the first part of the formula are repeated for every part which follows, with a further number being added each time to provide greater detail about the economic activity. The 5-digit 'sub-class' provides the SIC code, or, if the activity does not have a sub-class, the number 0 should be added to the digits in 'class' to provide the 5-digit SIC code.

Up to 4 SIC codes may be selected to describe the undertaking's economic activities.
